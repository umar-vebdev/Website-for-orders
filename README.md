# Website-for-Orders

Простой сайт для онлайн-заказа блюд с административной панелью для управления блюдами и заказами.  

## Стек технологий

- PHP 8.2  
- Laravel 12  
- MySQL  
- Blade (шаблоны фронтенда)  
- Tailwind CSS (стилизация)  

---

## Возможности проекта

### Для клиентов

- Просмотр меню (`/menu`)  
- Добавление блюд в корзину  
- Обновление и удаление блюд из корзины  
- Оформление заказа с указанием имени, телефона и адреса  
- Просмотр истории своих заказов (`/my-orders`)  
- Уникальный `client_id` в cookie для отслеживания заказов без регистрации  

### Для админов

- Вход через `/admin/login`  
- CRUD для блюд (создание, редактирование, удаление)  
- Просмотр всех заказов пользователей  
- Изменение статуса заказа (`new`, `processing`, `completed`, `canceled`)
